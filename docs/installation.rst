============
Installation
============

If you have ``virtualenvwrapper`` installed, at the command line type:

.. code-block:: bash

    mkvirtualenv pygogo
    pip install pygogo

Or, if you have ``virtualenv`` installed:

.. code-block:: bash

	virtualenv venv
	source venv/bin/activate
	pip install pygogo

Otherwise, you can install globally::

    pip install pygogo
